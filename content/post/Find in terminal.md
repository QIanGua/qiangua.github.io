# Find files from terminal

[TOC]

## Several Useful Commonds

目录搜索:
- find
- fd

文本搜索:
- grep
- rg

## Find files 
### find
```shell
> find root_path -name '*.ext'
> find root_path -name '*pattern_1*'
```
### fd

```shell
> fd 'file'
```

## Find Word in Files

### grep

```shell
> grep text file_name
```

### rg(ripgrep)

```shell
> rg 'word'
```
